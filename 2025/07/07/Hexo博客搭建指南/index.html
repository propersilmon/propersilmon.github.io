Hexo博客搭建指南

Posted on 2025-07-07 Edited on 2025-11-14 In 工具 Views:
Symbols count in article: 2k Reading time ≈ 7 mins.

Hexo是一个快速、简洁且高效的博客框架。有Node.js 所带来的超快生成速度。 上百个页面在几秒内完成渲染。Hexo 支持 GitHub Flavored Markdown 的所有功能，甚至可以整合 Octopress 的大多数插件。只需一条指令即可部署到 GitHub Pages, Heroku 或其他平台。并且其强大的 API 带来无限的可能。 支持数种模板引擎（EJS，Pug，Nunjucks等）。 可以与现有的NPM包 (Babel, PostCSS, Less/Sass 等) 轻松地集成。

字段名 说明
title 文章标题
date 创建时间（Hexo 自动生成）
updated 更新时间（可选）
tags 标签（可多个）
categories 分类（可多个，区别见下）
author 作者（可选）
permalink 自定义 URL
comments 是否启用评论（true/false）
top 是否置顶（某些主题支持）
cover 封面图（某些主题支持）
excerpt 自定义摘要（主题支持）

博客美化

更换主题

你可以在Themes | Hexo页面下载各种各样的主题。下载完成后，将文件解压缩到hexo项目下theme文件夹下。

然后修改站的配置文件_config.yml，修改字段theme为你刚下载的主题名称
